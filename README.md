# ☕ Pre-DSA 150 Questions (Java)

Welcome to the **Pre-DSA 150** repository! 🚀
This repository contains **150 carefully selected programming problems** focused on building a strong foundation in **Java and problem-solving skills** before diving into Data Structures & Algorithms (DSA).

---

## 📌 About This Repository

* 📚 Covers **basic to intermediate programming concepts**
* 💻 Implemented using **Java**
* 🎯 Ideal for:

  * Beginners in programming
  * Students preparing for placements
  * Anyone starting DSA journey

---

## 🧠 Topics Covered

* 🔢 Number Theory (Prime, GCD, LCM, Armstrong, etc.)
* 🔁 Loops & Recursion
* 🔤 String Manipulation
* 📊 Arrays & Matrix Operations
* 🧩 Pattern Problems
* 📈 Mathematical Computations
* 🧮 Sorting & Searching Basics
* 🧠 Logical & Problem-Solving Skills

## 🚀 How to Use

1. Clone the repository:

   ```bash

   ```

2. Navigate to the folder:

   ```bash
   cd pre-dsa-150
   ```

3. Compile and run Java files:

   ```bash
   javac FileName.java
   java FileName
   ```
